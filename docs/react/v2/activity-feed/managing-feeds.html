Managing Activity Feeds

In this section, we will show you how to manage activity feeds from the activity feeds service.

Prerequisites​

  1. App Initialized (see Initializing Busy Hour Project/App)
  2. Main Service Initialized (see Initializing Busy Hour Main Service)
  3. Activity Feed Service Initialized (see Initializing Activity Feeds)

Managing Activity Feeds​

To start managing your activity feeds (e.g. create, delete, etc.), you will need to use the useFeedManager hook which is exported from the @busy-hour/react/hooks or @busy-hour/react/hooks/v2 package. The hook then will return an object detailed as below:

PropertyTypeDescription
createFeedfunctionCreate a new activity feed
deleteFeedfunctionDelete an existing activity feed
likeFeedfunctionLike an activity feed
dislikeFeedfunctionDislike an activity feed

Creating Activity Feeds​

To create an activity feed, you can use the createFeed function which is one of the return properties of the useFeedManager hook. To use it, you need to pass some properties which detailed as below.

PropertyTypeDescription
titlestring (optional)The title of the activity feed
typefeed | string (optional)The type of the activity feed
descriptionstring (optional)The description of the activity feed
attachmentsAttachment[] (optional)Any file information for the activity feed
customobject (optional)Any custom information for the activity feed

Quick Example​

src/main.tsx
import { useFeedManager } from '@busy-hour/react/hooks';

export default function Main() {
  const { createFeed } = useFeedManager();

  const createNewFeed = () => {
    return createFeed({
      title: 'New activity feed',
      type: 'feed',
      description: 'This is a new activity feed',
      // Custom properties for the activity feed
      custom: {
        color: '#ff0000',
      },
    });
  };

  return <MainApp />;
}

Deleting Activity Feeds​

To delete an activity feed, you can use the deleteFeed function which is one of the return properties of the useFeedManager hook. To use it, you need to pass the feedId of the activity feed.

Quick Example​

src/main.tsx
import { useFeedManager } from '@busy-hour/react/hooks';

export default function Main() {
  const { deleteFeed } = useFeedManager();

  const deleteExistingFeed = () => {
    return deleteFeed('the-feed-id');
  };

  return <MainApp />;
}

Liking Activity Feeds​

To like an activity feed, you can use the likeFeed function which is one of the return properties of the useFeedManager hook. To use it, you need to pass the feedId of the activity feed and a liked boolean to determined to remove the like or not.

Quick Example​

src/main.tsx
import { useFeedManager } from '@busy-hour/react/hooks';

export default function Main() {
  const { likeFeed } = useFeedManager();

  const likeExistingFeed = () => {
    return likeFeed('the-feed-id', false); // Do not remove the like
  };

  return <MainApp />;
}

Disliking Activity Feeds​

To like an activity feed, you can use the dislikeFeed function which is one of the return properties of the useFeedManager hook. To use it, you need to pass the feedId of the activity feed and a liked boolean to determined to remove the dislike or not.

Quick Example​

src/main.tsx
import { useFeedManager } from '@busy-hour/react/hooks';

export default function Main() {
  const { dislikeFeed } = useFeedManager();

  const dislikeExistingFeed = () => {
    return dislikeFeed('the-feed-id', true); // remove the dislike
  };

  return <MainApp />;
}
