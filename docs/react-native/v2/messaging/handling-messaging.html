Handling Chat Messaging

In this section, we will show you how to handle chat messages from the messaging service.

Prerequisites​

  1. App Initialized (see Initializing Busy Hour Project/App)
  2. Main Service Initialized (seeInitializing Busy Hour Main Service)
  3. Messaging Service Initialized (see Initializing Messaging)

Handling Chat Messaging​

To start handling current/selected chats, you will need to use useChatMessaging hook which is exported from the @busy-hour/react-native/hooks or @busy-hour/react-native/hooks/v2 package. To use it, you just need to pass a config object which detailed below:

PropertyTypeDescription
chatIdstringChat id to retrieve messages
orderByasc | desc (optional)Order of the messages to retrieve (default: desc)
limitnumber (optional)Total message to retrieve (default: undefined)
isShouldListenboolean (optional)Determine if it should listen on mount or not (default: undefined)

Once the hook is called, it will return an object that contains messages and sendMessage. For more detail about the properties see the detail from this table:

PropertyTypeDescription
messagesmessage[]The messages in the chat
sendMessagefunctionSend a message to the chat
listenChatMessagesfunctionStart listening to chat messages
removeChatMessagesListenerfunctionStop listening to chat messages

Quick Example​

src/main.tsx

import React, { useEffect } from 'react'
import { useChatMessaging } from '@busy-hour/react-native/hooks'

export default function Main() {
  const { messages, sendMessage } = useChatMessaging({
    chatId: 'the-chat-id-to-listen-to',
    isShouldListen: true,
  })

  return <MainApp />
}

Sending Messages​

To send a message, you can use the sendMessage function which is one of the return properties of the useChatMessages hook. To use it, you need to pass some properties which detailed below:

PropertyTypeDescription
messagestring (optional)The message to send
attachmentsAttachment[] (optional)Attachment data to send
customobject (optional)The custom message to send
typemessagestring (optional)

Attachment Object​

PropertyTypeDescription
uristringURI of the attachment
namestringName of the attachment
typestringType of the attachment

Quick Example​

import { useChatMessaging } from '@busy-hour/react-native/hooks';

export default function Main() {
  const { sendMessage } = useChatMessaging({
    chatId: 'the-chat-id-to-listen-to',
  });

  const sendNewMessage = useCallback((payload: unknown) => {
    sendMessage(payload)
  }, [sendMessage]);

  return <MainApp />

}
