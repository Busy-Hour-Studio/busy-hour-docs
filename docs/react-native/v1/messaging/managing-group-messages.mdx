# Managing Group Messages

In this section, we will show you how to manage group messages from the messaging service.

## Prerequisites

<MessagingPrerequisites />

## Managing Group Messages

To start managing your group messages (e.g. `create`, `update`, `delete`, etc.), you will need to use the `useGroupManager` hook which is exported from the `@busy-hour/react-native/hooks/v1` package. To use it, you just need to call it like any other hook and it will return an object. For more detail about the properties see the detail from this table:

| Property        | Type       | Description                         |
| --------------- | ---------- | ----------------------------------- |
| `createGroup`   | `function` | Create a new group messages         |
| `updateGroup`   | `function` | Update an existing group messages   |
| `deleteGroup`   | `function` | Delete an existing group messages   |
| `addMembers`    | `function` | Add member to the group messages    |
| `removeMembers` | `function` | Remove member to the group messages |
| `leaveGroup`    | `function` | Leave from group messages           |

## Creating Group Messages

To create a group message, you can use the `createGroup` function which is one of the return properties of the `useGroupManager` hook. To use it, you need to pass some properties which detailed below.

| Property      | Type                | Description                                      |
| ------------- | ------------------- | ------------------------------------------------ |
| `avatar`      | `string` (optional) | The avatar of the group messages                 |
| `name`        | `string` (optional) | The name of the group messages                   |
| `description` | `string` (optional) | The description of the group messages            |
| `members`     | `Member[]`          | List of member to be added to the group messages |

### Member Object

| Property | Type        | Description                                        |
| -------- | ----------- | -------------------------------------------------- |
| `userId` | `string`    | User id to be added                                |
| `role`   | `GroupRole` | User role, import from `@busy-hour/react-native/enums/v1` |

### Quick Example

```tsx
import { useGroupManager } from '@busy-hour/react-native/hooks/v1';
import { GroupRole } from '@busy-hour/react-native/enums/v1';

export default function Main() {
  // highlight-next-line
  const { createGroup } = useGroupManager();

  const createGroupMessages = async () => {
    // highlight-start
    return createGroup({
      name: 'My Group',
      description: 'This is my group',
      members: [
        {
          userId: 'your-user-id',
          role: GroupRole.ADMIN,
        },
        {
          userId: 'other-user-id',
          role: GroupRole.MEMBER,
        },
      ],
    });
    // highlight-end
  };

  return <MainApp />;
}
```

## Updating Group Messages

To update a group message, you can use the `updateGroup` function which is one of the return properties of the `useGroupManager` hook. To use it, you need to pass some properties which detailed below.

| Property      | Type                | Description                           |
| ------------- | ------------------- | ------------------------------------- |
| `groupChatId` | `string`            | Group chat id to be updated           |
| `avatar`      | `string` (optional) | The avatar of the group messages      |
| `name`        | `string` (optional) | The name of the group messages        |
| `description` | `string` (optional) | The description of the group messages |

### Quick Example

```tsx
import { useGroupManager } from '@busy-hour/react-native/hooks/v1';

export default function Main() {
  // highlight-next-line
  const { updateGroup } = useGroupManager();

  const updateGroupMessages = async () => {
    // highlight-start
    return updateGroup('group-chat-id', {
      name: 'My new Group',
      description: 'This is my group',
    });
    // highlight-end
  };

  return <MainApp />;
}
```

## Deleting Group Messages

To delete a group message, you can use the `deleteGroup` function which is one of the return properties of the `useGroupManager` hook. To use it, you just need to pass the group `chatId` to delete.

### Quick Example

```tsx
import { useGroupManager } from '@busy-hour/react-native/hooks/v1';

export default function Main() {
  // highlight-next-line
  const { deleteGroup } = useGroupManager();

  const deleteGroupMessages = async () => {
    // highlight-next-line
    return deleteGroup('group-chat-id');
  };

  return <MainApp />;
}
```

## Adding Members

To add a member to a group message, you can use the `addMembers` function which is one of the return properties of the `useGroupManager` hook. To use it, you need to pass some properties which detailed below.

| Property      | Type       | Description                                      |
| ------------- | ---------- | ------------------------------------------------ |
| `groupChatId` | `string`   | Group chat id to be updated                      |
| `members`     | `Member[]` | List of member to be added to the group messages |

### Quick Example

```tsx
import { useGroupManager } from '@busy-hour/react-native/hooks/v1';
import { GroupRole } from '@busy-hour/react-native/enums/v1';

export default function Main() {
  // highlight-next-line
  const { addMembers } = useGroupManager();

  const addGroupMembers = async () => {
    // highlight-start
    return addMembers('group-chat-id', [
      {
        userId: 'other-user-id-1',
        role: GroupRole.MODERATOR,
      },
      {
        userId: 'other-user-id-2',
        role: GroupRole.MEMBER,
      },
    ]);
    // highlight-end
  };

  return <MainApp />;
}
```

## Removing Members

To remove a member to a group message, you can use the `removeMembers` function which is one of the return properties of the `useGroupManager` hook. To use it, you need to pass some properties which detailed below.

| Property      | Type       | Description                                                  |
| ------------- | ---------- | ------------------------------------------------------------ |
| `groupChatId` | `string`   | Group chat id to be updated                                  |
| `members`     | `string[]` | List of member user id to be removed from the group messages |

### Quick Example

```tsx
import { useGroupManager } from '@busy-hour/react-native/hooks/v1';

export default function Main() {
  // highlight-next-line
  const { removeMembers } = useGroupManager();

  const removeGroupMembers = async () => {
    // highlight-start
    return removeMembers('group-chat-id', [
      'other-user-id-1',
      'other-user-id-2',
    ]);
    // highlight-end
  };

  return <MainApp />;
}
```
